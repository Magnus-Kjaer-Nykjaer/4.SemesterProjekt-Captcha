Her er der et link til vores repository og giver mulighed for at hente vores projekt ned:
https://github.com/Magnus-Kjaer-Nykjaer/4.SemesterProjekt-Captcha 


1. Før du starter projektet i Visual Studio skal du først gå ind på mappen der hedder \4.SemesterProjekt-Captcha\RavenDB 


2. Når du er derinde skal du køre filen der hedder run.ps1 i powershell, dette kan gøres ved at højreklikke filen og trykke run in powershell. 


3. Når du har gjort dette skulle der meget gerne åbne en side i browseren dette er vores RavenDB og fanen kan bare lukkes.


4. Hvis der ikke er sat multipel startup projekt skal Klienten og de to API’er lige tilføjes til det.


5. Når du har startet vores RavenDB kan du så køre vores program i Visual Studio.
